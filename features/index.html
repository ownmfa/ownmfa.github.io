Features

  • Easily add support for MFA/2FA to your product with simple API calls.
  • Retain control of your user and authentication workflow. OwnMFA is not an OAuth service or identity provider.
  • No personally identifiable information to be kept in sync. We don't store any PII, only opaque identifiers.
  • Detailed event reporting and authentication tracking.
  • No additional messaging services to integrate. We do it all, including SMS and email.
  • No surprise, per-message billing. All charges are included in the monthly, per-identity fee.
  • Standards-compliant: No proprietary applications for users to install.
  • Blazing fast REST and gRPC APIs.
  • Integration so easy, it can be completed today!

Authentication Methods

  • SMS one-time password (OTP)
  • Email one-time password (OTP)
  • Software token (offline HMAC- and time-based HOTP/TOTP)
  • Google Authenticator soft token (offline HOTP/TOTP)
  • Microsoft Authenticator soft token (offline TOTP)
  • Pushover push token
  • Hardware token (offline HOTP/TOTP)
  • Backup codes
  • Security questions
  • More added regularly!

Benefits

  • Protect your business and your customers from credential and account theft with secure, API-backed logins.
  • Integrate with your signup flow or for existing users alike, without a painful user experience.

  • A wide variety of authentication options to support every use case, with more added regularly.
  • Offline support via software and hardware tokens that work even when SMS and voice aren't available.
